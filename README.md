📊 PhonePe Transaction Data Analysis Dashboard

This interactive Power BI dashboard analyzes transaction data from January 2024 to August 2024. The main objective of this dashboard is to monitor overall transaction performance, analyze successful and failed transactions, and identify key reasons behind payment failures.

🔎 Dashboard Overview:
🔹 Date Range Filter

A dynamic date slicer is provided at the top, allowing users to select a specific time period for analysis.

🔹 Key Performance Indicators (KPIs)

The dashboard displays important performance metrics:

Total Amount: 2,009.27M

Total Transactions: 180.054K

Successful Transactions: 172.89K

Failed Transactions: 6.799K

These KPIs give a quick summary of overall business performance.

🔹 Failed Payment Reason (Pie Chart)

The pie chart shows the distribution of successful and failed transactions.
Approximately 96% of transactions are successful, while a small percentage fails due to reasons such as:

Server Error

Insufficient Balance

Wrong PIN

Wrong Information

This helps identify operational issues affecting transaction success rates.

🔹 Service-wise Transaction Analysis (Bar Chart)

The bar chart compares transaction amounts across different services:

Loans

Insurance

Money Transfer

Recharge Bills

It clearly shows which service contributes the most revenue (Loans being the highest contributor).

🔹 Monthly Trend Analysis (Line Chart)

The line chart displays the monthly transaction amount trend from January to August.
This helps track growth patterns, seasonal fluctuations, and performance consistency over time.
